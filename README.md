# tron-contract

Configured with [TronBox](https://github.com/tronprotocol/tron-box).

### Install tronbox:

```npm install -g tronbox```

Add private key and public key tronbox.js file.
### Compile:

```tronbox compile```

### Migrate:

```tronbox migrate --reset --net deployment```
